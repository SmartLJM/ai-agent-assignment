# Benchmark Coverage

The benchmark contains 30 questions.

Question types included:

- short-answer
- true-false
- single-choice
- multiple-choice

Difficulty range: 2–6.

Covered drug-discovery areas include target identification, validation, hit discovery, lead optimization, docking, virtual screening, ADME, toxicology, PK/PD, repurposing, clinical trials, regulatory approval, AI/ML, ChEMBL, PubChem, QSAR, generative design, assay quality, knowledge graphs, and explainability.

Run:

```bash
python evaluate_benchmark.py
```
