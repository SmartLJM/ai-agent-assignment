# 1. 概述

问题元数据以 JSON 格式组织，每个问题保存为一个独立的 `.json` 文件，包含以下通用字段：

---

# 2. 通用字段

| 字段           | 类型      | 必填  | 说明                                                                  |
| ------------ | ------- | --- | ------------------------------------------------------------------- |
| `id`         | string  | 是   | 唯一标识，格式 `Q{NNNN}`，如 `Q0001`                                         |
| `question`   | string  | 是   | 问题完整文本，选择题的选项嵌入此字段（如 `"...\nA. ...\nB. ..."`）                       |
| `type`       | string  | 是   | `short-answer` / `true-false` / `single-choice` / `multiple-choice` |
| `difficulty` | integer | 是   | 难度 1–10                                                             |
| `answer`     | object  | 是   | 答案对象，结构随题型变化，详见 2.1                                                 |
| `sources`    | array   | 是   | 答案来源，详见 2.2                                                         |
| `theme`      | array   | 是   | 主题及关键词，详见 2.3                                                       |

## 2.1 `answer` — 答案对象

`answer` 是一个对象，包含两个必填字段 `type` 和 `content`，并按题型追加扩展字段。

### 字段速查

| `answer.type` | 适用题型 | `answer.content` 类型 | 扩展字段 |
| --- | --- | --- | --- |
| `exact` | 判断题 | boolean | — |
| `exact` | 单选题 | string（如 `"A"`） | — |
| `exact` | 多选题 | array（如 `["A","C"]`） | — |
| `similar` | 简答题 | string | — |
| `keypoint` | 简答题 | string | `keypoints` |
| `similar` / `keypoint` | 简答题（多跳推理） | string | `reasoning_steps` |

### 各字段说明

**`keypoints`** — 踩分点列表（仅 `type: "keypoint"` 时）

```json
"keypoints": [
  {"point": "踩分点描述", "score": 分值}
]
```

**`reasoning_steps`** — 推理步骤列表（多跳推理时）

```json
"reasoning_steps": [
  {
    "step": 1,
    "description": "该步推理目标",
    "source": {
      "knowledge_set_id": "...",
      "original_text": "..."
    }
  }
]
```

## 2.2 `sources` — 答案来源

```json
"sources": [
  {
    "knowledge_set_id": "知识集ID",
    "original_text": "引用原文片段"
  }
]
```

## 2.3 `theme` — 主题分类

列表，每个元素为 `{主题名: [关键词列表]}`，全部 kebab-case。

```json
"theme": [
  {"climate-science": ["greenhouse-effect", "radiative-forcing"]}
]
```

# 3. 完整示例

## 3.1 简答题 — 踩点给分

```json
{
  "id": "Q0001",
  "question": "温室效应的主要原理是什么？",
  "type": "short-answer",
  "difficulty": 4,
  "answer": {
    "type": "keypoint",
    "content": "大气中的温室气体（如CO₂、CH₄）吸收地表发出的长波辐射并重新向地面发射，导致地表温度升高。",
    "keypoints": [
      {"point": "温室气体吸收长波辐射", "score": 3},
      {"point": "重新向地面发射", "score": 3},
      {"point": "地表温度升高", "score": 4}
    ]
  },
  "sources": [
    {
      "knowledge_set_id": "encyclopedia_001",
      "original_text": "Greenhouse gases absorb and re-emit infrared radiation from Earth's surface..."
    }
  ],
  "theme": [
    {"climate-science": ["greenhouse-effect", "radiative-forcing", "climate"]}
  ]
}
```

## 3.2 多选题

```json
{
  "id": "Q0005",
  "question": "以下哪些是可再生能源？\nA. 太阳能\nB. 天然气\nC. 风能\nD. 石油",
  "type": "multiple-choice",
  "difficulty": 3,
  "answer": {
    "type": "exact",
    "content": ["A", "C"]
  },
  "sources": [
    {
      "knowledge_set_id": "encyclopedia_002",
      "original_text": "Renewable energy sources include solar, wind, hydro..."
    }
  ],
  "theme": [
    {"general": ["renewable-energy", "solar-energy", "wind-energy", "fossil-fuel"]}
  ]
}
```

## 3.3 多跳推理

```json
{
  "id": "Q0010",
  "question": "为什么赤道地区年温差小而内陆地区年温差大？",
  "type": "short-answer",
  "difficulty": 6,
  "answer": {
    "type": "similar",
    "content": "赤道地区全年太阳辐射量变化小，且海洋比热容大调节气温；内陆地区远离海洋缺乏调节，且纬度较高导致太阳辐射季节差异大。",
    "reasoning_steps": [
      {
        "step": 1,
        "description": "赤道地区太阳辐射的季节变化特点",
        "source": {
          "knowledge_set_id": "textbook_geography_001",
          "original_text": "Solar radiation at the equator remains relatively constant year-round..."
        }
      },
      {
        "step": 2,
        "description": "海洋和陆地的比热容差异对气温的调节作用",
        "source": {
          "knowledge_set_id": "textbook_geography_001",
          "original_text": "Water has a higher specific heat capacity than land, moderating coastal temperatures..."
        }
      },
      {
        "step": 3,
        "description": "内陆地区远离海洋导致缺乏气温调节",
        "source": {
          "knowledge_set_id": "textbook_geography_001",
          "original_text": "Inland regions lack the moderating influence of oceans, leading to greater temperature extremes..."
        }
      }
    ]
  },
  "sources": [
    {
      "knowledge_set_id": "textbook_geography_001",
      "original_text": "Equatorial regions experience minimal temperature variation..."
    }
  ],
  "theme": [
    {"general": ["latitude", "solar-radiation", "climate"]},
    {"oceanography": ["ocean", "heat-capacity"]}
  ]
}
```

# 4. 文件组织

```
基础模块/
└── benchmark/
    ├── Q0001.json
    ├── Q0002.json
    └── ...
```

> `knowledge_set_id` 应与知识集目录中的文件夹名称保持一致。
