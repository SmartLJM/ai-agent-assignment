# Ligand-Based Drug Design

**Knowledge set ID:** `dd-007-ligand-based-design`  
**Theme:** `drug-discovery`  
**Source:** Computational methods in drug discovery  

## Curated Content

Ligand-based drug design uses knowledge from molecules already known to bind or modulate a target.
It is valuable when target structures are unavailable or uncertain. Common methods include molecular
similarity search, pharmacophore modeling, quantitative structure-activity relationship modeling,
matched molecular pair analysis, and machine-learning prediction.  The central assumption is that
compounds with similar structures or shared pharmacophoric features may show related biological
activity. This assumption is useful but imperfect because small structural changes can cause
activity cliffs, altered selectivity, or different ADMET behavior.  Ligand-based methods should be
validated with held-out data and prospective experiments. A good answer should distinguish ligand-
based design from structure-based design: ligand-based design starts from known active compounds,
whereas structure-based design starts from the target’s three-dimensional binding site.
