# 课程作业评分标准

## 1. 总体评分

总分为**100分**，由三个部分组成：

| 模块 | 满分 | 说明 |
|------|------|------|
| **知识资产** | 35分 | 独立评分 |
| **评测基准** | 35分 | 独立评分 |
| **知识问答项目** | 30分 | 来自可选模块的得分 |

**总分 = 知识资产得分 + 评测基准得分 + 知识问答项目得分**

---

## 2. 知识资产模块（35分）

### 2.1 基础要求（21分）
| 要求       | 满分  | 说明                                                |
| -------- | --- | ------------------------------------------------- |
| **数量达标** | 7分  | 至少20个有效知识集（每个含content/、keywords.json、source.json） |
| **文件完整** | 7分  | 每个知识集三个文件齐全，无缺失                                   |
| **格式规范** | 7分  | 全部使用kebab-case格式，无格式错误                            |

### 2.2 质量要求（14分）
| 要求 | 满分 | 说明 |
|------|------|------|
| **领域内广度** | 5分 | 在选定领域内覆盖不同类型（如：计算实验、湿实验、综述、领域介绍等） |
| **内容充实** | 5分 | 知识集内容完整，非空壳文件 |
| **来源可靠** | 4分 | 学术论文有DOI，网页资源有有效URL |

**注**：有效知识集指内容充实、来源可查、符合规范的知识集。

---

## 3. 评测基准模块（35分）

### 3.1 基础要求（21分）
| 要求 | 满分 | 说明 |
|------|------|------|
| **问题数量** | 7分 | 至少20个有效问题 |
| **答案准确** | 7分 | 所有问题答案正确 |
| **格式规范** | 7分 | JSON格式正确，必填字段齐全 |

### 3.2 质量要求（14分）
| 要求 | 满分 | 说明 |
|------|------|------|
| **问题质量** | 5分 | 问题表述清晰，无歧义 |
| **来源引用** | 5分 | 每个问题有准确的来源引用 |
| **主题分类** | 4分 | 主题分类正确，使用预定义主题词 |

**注**：有效问题指能清晰回答、有准确来源、格式正确的问题。

---

## 4. 知识问答项目（30分，来自可选模块）

### 4.1 可选模块选择

学生从可选模块中选择若干任务，每个任务有**难度值i**（1=基础, 2=中等, 3=挑战）**，要求选择的难度值累计为4**。

### 4.2 评分计算
1. **每个模块单独评分**（0-100分），主要看：
   - 项目能否跑通
   - 能否通过评测基准测试
   - 是否展示了运行结果

2. **权重计算**：
   ```
   总难度和 = 所选所有模块难度值之和
   某模块权重 = 该模块难度值 / 总难度和
   ```

3. **知识问答项目得分**：
   ```
   加权总分 = Σ(模块得分 × 该模块权重)
   知识问答项目最终得分 = 加权总分 × 0.3
   ```

### 4.3 示例
学生选择了：
- 模块A（难度2，得分80分）
- 模块B（难度1，得分70分）
- 模块C（难度3，得分90分）

计算：
- 总难度和 = 2 + 1 + 3 = 6
- 模块A权重 = 2/6 ≈ 0.333
- 模块B权重 = 1/6 ≈ 0.167  
- 模块C权重 = 3/6 = 0.5
- 加权总分 = 80×0.333 + 70×0.167 + 90×0.5 = 26.64 + 11.69 + 45 = 83.33分
- 知识问答项目得分 = 83.33 × 0.3 = 25.0分

---

## 5. 可选模块难度说明

### 5.1 难度1（基础）
- 实现基础功能
- 跑通评测基准
- 展示基本结果

### 5.2 难度2（中等）  
- 在基础上有所优化
- 处理更多边界情况
- 结果分析更深入

### 5.3 难度3（挑战）
- 有技术挑战性
- 实现创新点
- 结果有深度分析

---

## 6. 总分计算示例

假设某学生：
- 知识资产模块：30分（共35分）
- 评测基准模块：28分（共35分）
- 知识问答项目：25分（加权总分83.33 × 0.3）

**总分 = 30 + 28 + 25 = 83分**
