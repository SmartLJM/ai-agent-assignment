# Large files omitted

This repository copy includes source code and lightweight documentation only.

The following local artifact categories were intentionally not committed:

- Python virtual environments, dependency folders, caches, and nested `.git` directories
- Large datasets and vector stores under `data/`, `storage/`, `npc_dataset_nii/`, and `faiss_index/`
- Model weights and binary model artifacts such as `.pth`, `.pt`, `.ckpt`, `.onnx`, and `.safetensors`
- Large arrays and medical image files such as `.npy`, `.npz`, `.nii`, `.nii.gz`, and `.mha`
- Videos and archives such as `.mp4`, `.mov`, `.zip`, `.rar`, `.7z`, and `.tar`
- Any file larger than 100 MB during the sync

The omitted files remain in the original local submission directory.
