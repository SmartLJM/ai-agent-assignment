# Virtual Screening

**Knowledge set ID:** `dd-009-virtual-screening`  
**Theme:** `drug-discovery`  
**Source:** Computational methods in drug discovery  

## Curated Content

Virtual screening uses computational methods to prioritize compounds from chemical libraries before
experimental testing. Structure-based virtual screening docks compounds into a binding site, whereas
ligand-based virtual screening uses similarity, pharmacophores, QSAR, or machine-learning models
based on known active compounds.  The purpose is enrichment: increasing the proportion of useful
compounds in a smaller set that will be purchased or synthesized and tested. Virtual screening does
not replace experiments because prediction uncertainty, library bias, and model artifacts can
produce false positives and false negatives.  Modern workflows can combine filters for
physicochemical properties, PAINS alerts, novelty, synthetic accessibility, docking, machine
learning, and active learning. An agent should describe virtual screening as a prioritization tool,
not as definitive evidence of therapeutic efficacy.
